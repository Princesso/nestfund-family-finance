# NestFund Family Finance

NestFund is a beautiful, privacy-first web app to help partners manage incomes, expenses, debts, and investments together. It also includes friendly bookkeeping lessons for parents and kid-focused money activities.

## Features
- Track multiple incomes for each partner
- Fixed and variable expenses with monthly planning
- Debt overview with weighted APR and payoff priority
- Investment suggestions for S&P ETFs, bond funds, and real estate ETFs
- Log and track investments you have already made
- Learning modules for parents and kids with progress tracking
- Local-only storage using your browser's localStorage

## Tech Stack
- HTML5 + Tailwind CSS (CDN)
- jQuery 3.7.x
- Modular JavaScript with a single App namespace

## Getting Started
1. Open index.html for the marketing site.
2. Click "Open the App" to go to app.html.
3. Start by adding incomes and expenses. Use the Dashboard to pick your month and set savings and investment targets.

## Data Persistence
All your data is saved locally in your browser. To reset, clear your browser storage for this site.

## Accessibility
Keyboard friendly and designed for good contrast with Space Grotesk headlines and Inter body text. Animations respect reduced motion preferences.

## Disclaimer
This app is for educational and planning purposes only and does not provide financial advice.
